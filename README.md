# MultiSearch-OSINT
Un Gui qui permet de faire plusieurs recherches à la fois sur un thème qui se sélectionne au début.
Copier le code dans PROJET FINAL, et copier le dans votre compilateur de code en python.
Vous aurez besoin de telecharger customtkinter pour faire apparaitre le Gui (pip install customtkinter).
Projet codé sans ia sauf pour apprendre des commandes difficiles spécifiques.
